LinkedIn Automation – Go + Rod (PoC)

⚠️ **For educational purposes only.** Do not use on live accounts.

## Objective
Demonstrate human-like LinkedIn automation with Go and Rod.

## Features
- Manual login → continue automation
- Navigate Feed → Messaging
- Scroll conversations, open first chat, read last message
- Human-like delays, hovering, scrolling

## Setup
git clone <repo-link>
cd linkedin-automation
go mod tidy
go run main.go
Login manually in browser, press ENTER in terminal to continue
